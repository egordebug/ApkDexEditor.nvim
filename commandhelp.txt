session files:
  ~/.local/share/nvim/ade_session.txt     (for APK)
  ~/.local/share/nvim/ade_smali_session.txt (for Dex/Class/Java → smali)

1. :ApkOpen DexEditor <action-or-apk-path>

   :ApkOpen DexEditor ~/myapp.apk
       → decode APK with apktool -f
       → save session
       → auto run :ApkOpen DexEditor List

   :ApkOpen DexEditor Compile
       → (re)compile ALL smali* folders → classes.dex / classes2.dex / ...
       → create mod_work.apk → inject all .dex → replace original APK
       → keep original as .bak if not exists
       → notify success

   :ApkOpen DexEditor Back
       → close session
       → rm -rf temp folder
       → remove state file
       → cd back to _G.ADE_State.MainProjectPath (if set)
       → notify "APK session closed"

   :ApkOpen DexEditor List
       → show UI select with all smali* folders (smali, smali_classes2, ...)
       → on choice: set target_dex, save session, cd into folder
       → reopen Neotree float on that folder (or netrw if no neo-tree)

2. Single-file smali workflows

   :Dex2Smali <path-to.dex>
   :Class2Smali <path-to.class>
   :Java2Smali <path-to.java>

       → close previous session
       → create temp dir
       → convert to DEX if needed (javac+d8 / d8 / copy)
       → baksmali → /out folder
       → save session (only for Dex type currently)
       → cd + open Neotree/Explore on smali root

   Common subcommands (when session active):

   :Dex2Smali Compile    (also Class2Smali Compile, Java2Smali Compile)
       → smali → classes.dex
       → for Dex   → copy back to original .dex
       → for Class → dex2jar → .class (approximate!)
       → for Java  → warning: impossible to fully reverse to source

   :Dex2Smali Back       (same for Class2Smali / Java2Smali)
       → close session
       → rm -rf temp dir (base dir, not only /out)
       → remove state file
       → cd back to MainProjectPath

   :Dex2Smali List       (same for others)
       → cd to smali root
       → open Neotree float or netrw

   :Dex2Smali Search <pattern>
       → :vimgrep /pattern/j **/*.smali
       → :copen quickfix

   :Dex2Smali SearchFile <pattern>   → not implemented yet

3. ADB helpers

   :AdbInstall [optional-apk]
       → default: install current session APK
       → or specified file
       → uses adb install -r

   :AdbPush <local-file> [remote-path]
       → default remote: /data/local/tmp/
       → adb push ...

4. Terminal

   <C-t>    (normal & terminal mode)
       → toggle floating terminal (80% size)
       → auto cd into active session folder (APK root or smali root)
       → starts sh/zsh/bash


Examples:

Full APK edit:
1. :ApkOpen DexEditor ~/modifications/modded.apk
2. Edit smali_classes2/**/...
3. :ApkOpen DexEditor List -> switch to another DEX
4. :ApkOpen DexEditor Compile
5. :AdbInstall
6. :ApkOpen DexEditor Back

Quick single dex patch:
1. :Dex2Smali classes.dex
2. Edit ...
3. :Dex2Smali Compile

Session auto-restore:
• Quit Neovim during edit → reopen → see prompt to restore / delete
